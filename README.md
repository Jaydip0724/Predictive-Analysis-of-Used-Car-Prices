# Predictive-Analysis-of-Used-Car-Prices
This project aims to predict used car prices using machine learning by analyzing features such as brand, year, mileage, and location. It involves data cleaning, EDA, feature selection, and regression modeling to deliver accurate price predictions.

The project demonstrates the complete data science workflow ‚Äî from data preprocessing and exploratory data analysis (EDA) to feature selection, model building, and performance evaluation.
